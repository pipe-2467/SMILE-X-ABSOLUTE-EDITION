# 💀 Smile-X Absolute Edition
**Developer:** Pipe
**Concept:** รัน ServerScript อิสระทุกลูกด้วย Identity 8 Bypass

### วิธีติดตั้ง
1. อัปโหลดโฟลเดอร์ `jni/` และ `app/` ขึ้น GitHub
2. ใช้ Android Studio หรือ NDK ในการคอมไพล์เป็น APK
3. รันในแมพเพื่อ "อัดบาเรีย" ผ่านช่องโหว่ RemoteEvent

### ระบบ Spidy Bot
- เชื่อมต่อ Webhook อัตโนมัติ รายงานผลทุกการเจาะ
